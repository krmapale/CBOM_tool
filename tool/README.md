# CycloneDX_PQC

Tool for creating PQC focused CBOM.


## Help
```text
$ pqcbom -h

Usage: 
    
    pqcbom [OPTION] [ARGUMENT]

Options:

    -i, --input                     Input directory path 
    -o, --output                    Output file name
    -h, --help                      Show help
    -v, --version                   Show version number

```
